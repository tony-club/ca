Responsive Canvas Without Breaking Physics

Keep logic resolution fixed; scale visuals with CSS.

Published 2025-08-22

Run game physics at a constant internal resolution (e.g., 900×520). Scale the canvas's display size with CSS (width: 100%) for different devices. This approach keeps game logic simple and predictable, as speeds and distances are always consistent. Always clamp entity positions to the internal width/height to avoid out-of-bounds issues. For crisp visuals on high-DPI screens, you can render to a larger offscreen buffer and scale it down, but separating logic from display size is the key.
